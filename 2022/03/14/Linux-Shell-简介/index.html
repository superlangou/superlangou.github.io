Linux Shell 简介

发表于2022-03-14更新于2022-03-14

基础指令

pwd - 输出工作目录
cd - 切换工作目录

重要的文件名称：

~ - 家（home）目录
~ab - 用户“ab”的家目录
. - 当前目录
.. - 父目录（当前目录上一级目录）
/ - 根目录（最上一级目录，无父目录）

手册页（man pages）

man 查询的命令
大部分命令可以加 –help 或者 -h 来查询相关命令的使用帮助
如果想查询更多man命令的信息可以输入： man man

管理文件 删除 移动 创造和删除

cp 文件路径 目的路径 - 复制文件
mv 文件路径 目的路径 - 移动和重命名文件
rm 文件名称 - 永久删除文件
rmdir 目录 - 创建目录
touch 文件 - 创建一个空白文件
列出当前目录的文件:
ls 路径 - 列出文件
tree 路径 - 递归列出文件

隐藏和临时文件

隐藏文件以 . 开头，只有使用 -all 时才会出现

Tar

Linux中的压缩文件
标志：
-c 创建一个压缩包
-x 打开压缩包
-z 使用gipz压缩
-v 详细模式（显示进度）
-f 指定文件名

文本处理

sort 对文本文件的行进行排序
-d 按照字排序 -f 忽略大小写 -n 数字排序 -r 颠倒
cat 连接输入并在屏幕上输出
uniq 报告或省略重复的行
head/tail 输出第一/最后一行
-dx 输出第一/最后一个x字符 -nx：输出第一/最后x行

文件定向 ？

Grep（Global Regular Expression Print，全局正则表达式版本）

grep 为在文件中的搜索模式 [如果未提供文件，则以递归方式搜索所有文件]

标志

-c ？
-n 显示匹配项和行号
-h 显示没有文件名的匹配项
-e exp ？
-i 忽略大小写
-f file ？
-o 仅输出符合匹配项的行
-l 之显示文件名列表
-r 以递归的方式读取每个目录下所有文件

Pipes（管道）

Pipes是一种将一个命令的输出与输入链接到另一个命令的方法
grep compute 路径 - 在目标路径中寻找compute
